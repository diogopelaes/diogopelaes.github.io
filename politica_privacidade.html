Política de Privacidade para GPTs

Esta Política de Privacidade descreve como tratamos as informações coletadas e processadas por sistemas baseados em GPT (Generative Pretrained Transformers), usados para interações e processamento de linguagem natural. Entendemos a importância de proteger a privacidade dos usuários e garantir que suas informações sejam tratadas de maneira responsável e transparente.

1. Informações Coletadas

Os sistemas GPT podem coletar dois tipos principais de informações durante suas interações:

  • Informações fornecidas pelo usuário: Isso inclui texto inserido diretamente pelos usuários durante a interação com o sistema, como perguntas, comentários ou dados fornecidos no contexto de uma conversa.
  • Informações sobre a interação: Incluem dados técnicos, como registros de tempo de interação, preferências do usuário, e a forma como o sistema responde a diferentes solicitações.

2. Uso das Informações

As informações fornecidas durante as interações são usadas exclusivamente para:

  • Fornecer respostas adequadas: O sistema utiliza os dados inseridos para gerar respostas contextuais e relevantes para cada interação.
  • Aprimorar o desempenho: Dados anonimizados podem ser usados para ajustar e melhorar a qualidade e precisão dos modelos GPT, para garantir que o sistema seja mais eficaz ao responder às perguntas.
  • Desenvolver novos recursos: Podemos analisar as interações anonimizadas para entender melhor as necessidades dos usuários e desenvolver novas funcionalidades ou melhorias.

3. Compartilhamento de Informações

Os dados processados pelos GPTs não são compartilhados com terceiros, exceto nas seguintes situações:

  • Requisitos legais: Quando necessário, para cumprir com a lei, uma intimação ou outra demanda governamental.
  • Segurança e proteção de direitos: Se necessário para proteger os direitos, segurança ou propriedade do usuário, ou para investigar uma violação dos Termos de Uso.
  • Melhorias do sistema: Dados anonimizados podem ser compartilhados internamente para desenvolvimento de novos produtos ou melhorias, mas sempre sem revelar informações pessoais.

4. Retenção de Informações

As interações não são armazenadas indefinidamente. As informações fornecidas durante o uso dos GPTs são geralmente armazenadas por um período limitado, apenas o necessário para cumprir com os propósitos descritos acima, e depois são excluídas ou anonimizadas.

5. Segurança das Informações

Os GPTs utilizam várias medidas de segurança para proteger as informações dos usuários contra acesso, uso ou divulgação não autorizados. Essas medidas incluem criptografia de dados, controles de acesso restritos e monitoramento de atividades suspeitas.

6. Direitos do Usuário

Os usuários têm o direito de solicitar:

  • Acesso e correção: Solicitar uma cópia das suas interações ou pedir para corrigir informações imprecisas.
  • Exclusão de dados: Exigir a exclusão de suas interações armazenadas, desde que não sejam mais necessárias para os fins descritos nesta política.
  • Limitação do processamento: Restringir o uso de seus dados, especialmente para propósitos não essenciais.

7. Uso de Cookies

Nos casos em que GPTs são usados em plataformas online, cookies podem ser utilizados para:

  • Personalização: Lembrar preferências do usuário para futuras interações.
  • Aprimoramento de desempenho: Monitorar e otimizar o desempenho do sistema durante o uso.

8. Alterações a esta Política

Esta Política de Privacidade pode ser revisada de tempos em tempos. Quaisquer alterações significativas serão notificadas aos usuários através dos canais apropriados. O uso contínuo do sistema após tais mudanças constitui aceitação das revisões.

9. Contato

Para dúvidas ou preocupações relacionadas a esta Política de Privacidade, os usuários podem entrar em contato através do e-mail contato@gptsystem.com.
